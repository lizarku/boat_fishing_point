2. 추천 순위
================================================================================

[1순위] 스즈키 DT15AS
  - 장점: 33kg 동급 최경량, 세단 트렁크 적재 최적
  - 단점: 2020년경 단종, 부품 수급 점차 어려워질 수 있음
  - 중고 예상가: 150~200만원

[2순위] 야마하 15FMHS
  - 장점: 아직 신품 판매중, AS/부품 수급 우수, 내구성 검증
  - 단점: 36kg (스즈키보다 3kg 무거움)
  - 중고 예상가: 150~220만원

[3순위] 토하츠 M15D2S / 머큐리 15 Super
  - 장점: 신뢰성 높음
  - 단점: 41kg으로 휴대성 불리
  - 중고 예상가: 130~180만원

※ 부품 수급 걱정되면 야마하가 더 안전한 선택

3. 2행정 선외기 수명 및 내구성
================================================================================

[평균 수명]
- 2행정 선외기: 1,500 ~ 2,000시간
- 4행정 선외기: 3,000 ~ 4,000시간
- 레저용 연간 사용량: 50~100시간
- 10년 사용 시 예상 누적: 500~1,000시간

→ 10년된 엔진도 관리 잘 됐으면 충분히 사용 가능!

--------------------------------------------------------------------------------
연식보다 "관리 상태"가 더 중요!
--------------------------------------------------------------------------------

[좋은 상태 징후]
- 매 사용 후 담수 세척 이력
- 정기적인 오일 교환 (연 1회 또는 100시간마다)
- 꾸준히 사용됨 (연 50시간 이상)
- 샵에서 점검/관리 이력 있음
- 외관 깨끗, 부식 없음

[나쁜 상태 징후]
- 해수 사용 후 방치
- 오일 관리 안 함
- 몇 달~몇 년 방치된 이력
- 외관 부식/녹 심함
- 오일/연료 누출 흔적

※ "500시간 사용된 5년 엔진이 20시간만 사용된 5년 엔진보다 더 신뢰할 수 있다"

--------------------------------------------------------------------------------
2행정 특유 문제: 카본 축적
--------------------------------------------------------------------------------
- 2행정은 연소 과정에서 카본이 쌓임
- 관리 안하면 1,000시간도 못 갈 수 있음
- 확인: 점화플러그 상태 (검게 그을린 정도)
- 해결: 디카본(decarbonizing) 처리

5. 중고 구매 전 필수 체크리스트
================================================================================

□ 샤프트 길이 확인
  - 반드시 "S" (숏샤프트, 15인치) 확인
  - 고무보트 트랜섬 높이: 보통 15인치

□ 압축 테스트
  - 두 실린더 압축값 측정
  - 정상: 두 실린더 차이 10% 이내
  - 낮은 압축 = 피스톤링/실린더 마모

□ 냉각수 배출 확인
  - 시동 걸고 냉각수 정상 배출되는지 확인
  - 안 나오면 = 임펠러 고장 또는 막힘

□ 기어오일 상태
  - 드레인 플러그로 오일 확인
  - 정상: 검은색/갈색 오일
  - 불량: 우유색 (수분 침투) → 씰 교체 필요

□ 외관 상태
  - 부식/녹 정도 확인
  - 오일 누출 흔적
  - 연료 누출 흔적
  - 프로펠러 상태

□ 시동 테스트
  - 한번에 시동 걸리는지
  - 공회전 안정성 (떨림/꺼짐 없는지)
  - 가속 시 이상 소음 없는지

□ 사용 이력 확인
  - 대략적인 사용 시간 (아워미터 있으면 확인)
  - 주로 민물/바다 중 어디서 사용했는지
  - 정비 이력 (오일교환, 임펠러 교체 등)

7. 가격 협상 포인트
================================================================================

[가격 낮출 수 있는 요소]
- 압축 테스트 결과 낮음
- 임펠러 교체 필요 (부품비 3~5만원 + 공임)
- 기어오일에 수분 (씰 교체 10~20만원)
- 외관 부식 심함
- 시동 불량/불안정

[적정 가격 판단]
- 신품 대비 50% 이하면 양호
- 10년 이상: 신품의 30~40%
- 5~10년: 신품의 40~50%
- 5년 미만: 신품의 50~60%

8. 구매 후 필수 정비
================================================================================

중고 구매 후 바로 해야 할 정비:

1. 기어오일 교환 (~2만원)
2. 임펠러 점검/교체 (~5만원)
3. 점화플러그 교체 (~1만원)
4. 연료필터 점검/교체
5. 연료라인 점검
6. 프로펠러 점검 (휘어짐, 손상)

※ 총 예상 비용: 10~15만원 (직접 정비 시)
※ 샵 의뢰 시: 20~30만원
