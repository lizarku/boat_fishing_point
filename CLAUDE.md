# 보트낚시 프로젝트 지침

## 낚시 기록 템플릿

```
[지깅/트롤링/드리프트] @ [위치]
• 순서: 1.[동작] 2.[동작] 3.[입질대응]
• 바람+조류: [현재상황] → [대응]
• ⚠️ 복귀: 일몰 1시간 전 필수 출발
```

---

## 바다타임 URL 형식

```
https://www.badatime.com/[번호]/[유형]/[날짜]

유형: tide(조석), hour-cast(시간별날씨), sea-temp(수온), sunrise-set(일출일몰)
```

## 국립해양조사원 API

```
http://www.khoa.go.kr/api/oceangrid/tideObsPreTab/search.do
?ServiceKey=wo0t70RrbJvdhG6zu53NNQ==
&ObsCode=DT_0017(태안)

```

## 검색 팁

- 지역번호: 구글 드라이브 "바다타임 지역번호" 검색
- 어종 시즌: "[지역] [월] 낚시 어종"
- 포인트: "[지역] 보트낚시 포인트"
- 금어기: "해양수산부 금어기 금지체장"
